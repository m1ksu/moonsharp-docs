Enum FuzzySymbolMatchingBehavior

A flag that controls if/how symbols (method, property, userdata) are fuzzily matched when they do not exist. Flags can be combined for multiple checks.

Namespace: MoonSharp.Interpreter
Assembly: MoonSharp.Interpreter.dll
Syntax
public enum FuzzySymbolMatchingBehavior : int

Fields

Name Description
Camelify

Underscores in symbols are converted to camelcase. For example, test_method() becomes testMethod()
None

No fuzzy matching is performed.
PascalCase

Converts a symbol to pascal case. For example, test_Method_two() becomes TestMethodTwo()
UpperFirstLetter

The first letter of a symbol will be uppercased (to check for common C# naming conventions). For example, testMethod() becomes TestMethod()
