Class AsyncExtensions

This class contains extension methods providing async wrappers of many methods. Asynchronous execution is performed by scheduling the method on the thread pool (through a Task.Factory.StartNew).

This type is supported only on .NET 4.x and .NET 4.x PCL targets.
